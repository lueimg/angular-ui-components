Displays when a list of options need to be presented to the user.
Use when a single option needs to be selected from a list.
Click the actionable text to open the menu. Clicking a selection will apply that option and close the menu. States will mirror the Text Field pattern, where applicable.

Basic Usage:

By default the select is a block level element and will fit to the width of it's container. A selected option who's content exceeds the width of the container will be redacted with an ellipsis.

Dynamic Size:

Optionally the input [dynamicSize]="true" can be provided and the select element will grow/shrink to fit the content length of the selected option.

Labels:

A label can be provided through the [label] input. The label can be positioned to either 'top' or 'left' through the [labelPosition] input. By default the label will only be as large as it's content, however the [labelWidth] input can be used to set a specific value
