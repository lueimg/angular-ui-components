Displays as the standard text field within page content and when there are many input fields present.

Font colors may be changed for specific usage.

Useful for scenarios where you expect the user to input long or mulit-line text.

Label and color transforms based on the state

Basic Usage

By default the text area is a block level element. Set the desired number of rows using the "rows" Input, default is 2.

Code snippet:

Auto Resize

Optionally, the input will automatically resize itself as the user types. Use attribute "autoresize" to activate. The input will be sized intially based on the [rows] input and will autoresize until the [maxRows] input has been reached. The default value for [rows] and [maxRows] is 2.

Code snippet:

Manual Resize

The normal CSS rules for the "resize" can be applied using the [resize] input.
The acceptable values are "none", "horizontal", "vertical", "inherit", "initial", or "both".
Default value is "none".
Note: Using "horiztonal" or "both" will result in the component becoming an "inline-block" element.
