Displays a pulse dot clickable which will show an overlay layer with a title and message

Available positions: [position]= place-right, place-left, place-top, place-bottom

Available size: [size] = '' , 'large'

Use near any element or component

Click on "Don't show me tips will save a local storage item by CoachTipsService"

Example:

<ui-coach-tip title="Test Title" message="Test Message." size="large" ></ui-coach-tip>
