Wizard

Displays a component with a Navigation at the top and a next button at the bottom

Each step compoennt require to be added as EntryComponent

Properties

Property Type Default Options Description
title string '' any string Label to the left of the navigation
components {{ '{' }} {{ '}' }}: WizardComponents null Object base on WizardComponents interface.
It required the properties: onEmpty: any; steps: any[]; onComplete: any;
